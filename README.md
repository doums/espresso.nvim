## Espresso.nvim

:coffee: A [nvim](https://neovim.io/) colorscheme

### Install

As usual use your plugin manager

```lua
require('paq')({
  -- ...
  'doums/espresso.nvim',
})
```

Then load the colorscheme in your config

```lua
vim.cmd('colorscheme espresso')
```
